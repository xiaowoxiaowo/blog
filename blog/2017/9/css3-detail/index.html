CSS3中的一些冷门属性
29 September 2017
CSS

对于知识的全面了解是一个程序员的优良品质，CSS对于我们前端来说是最常见的一种语言。但是这种常见的语言在不同的人手里，却可能有着 手枪和大炮的差距。对于CSS冷门属性的了解，也是我们应该学习的内容。

我会重新全面的看一遍CSS3，CSS2的所有属性内容，列出一些对我来说可能不常用，但是也有其发挥空间的属性。

animation-play-state

CSS3动画的一个属性，其实学习动画的时候也接触过，简单的属性配置便能够使CSS3动画暂停或者继续。感觉用处很大。

animation-direction

也是CSS3动画的一个属性，检索或设置对象动画在循环中是否反向运动，在设置循环动画的时候很有用

background-clip

指定对象的背景图像向外裁剪的区域,其实就是设置背景图片展示包不包括padding以及margin这类距离，个人感觉用处不大

background-origin

设置或检索对象的背景图像计算background-position时的参考原点(位置)。跟上面一个类似，设置背景图片显示的位置，但是平时用background-position 比这个方便多了。

font-variant

设置小型大写字母的字体显示文本，这意味着所有的小写字母均会被转换为大写，但是所有使用小型大写字体的字母与其余文本相比，其字体尺寸更小

counter-increment,counter-reset

这两个是一起用的属性, 需要用在:before 以及 :after的content里面，可以自动计算出现标签的次数，引出数字。算是少见的CSS可以动态产生变量的属性

list-style

这里面有三个不常用的属性list-style-image,list-style-position,list-style-type。设置li前缀的图片可以通过list-style-image来设置, list-style-position设置列表项标记的放置位置，list-style-type设置列表项标记的类型，有很多的类型可以选择

column-count

指定一个标签里的文字应该分为的列数

column-gap

指定列之间的差距

column-rule

这里面有三个属性,column-rule-color指定列之间的颜色规则,column-rule-style 指定列之间的样式规则,column-rule-width 指定列之间的宽度规则

column-width

指定列的宽度,分成多少列由宽度决定

column-span

指定元素应该跨越多少列，这是需要跟另外一个column去对比，可以设置列数的占比

columns

缩写属性，设置列宽和列数

clip

可以对绝对定位的元素进行剪裁

direction

设置文本显示的左右顺序

text-transform

设置一段文字是否首字母大写，文字是否全部大写或小写

word-wrap

设置浏览器是否对过长的单词进行换行

PS:省略了很多没有浏览器支持或者支持性特别差的属性。
