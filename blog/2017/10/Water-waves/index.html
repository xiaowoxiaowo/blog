CSS实现的水波荡漾效果
13 October 2017
CSS

国庆中秋双节8天假期过后,人又从慵懒的假期转变成工作的状态，就这样到了第一个星期的周五，突然就不想工作了，还是找点内容写个博客。

今天介绍一个纯CSS实现的水波荡漾效果，简单易用，效果很好。我们先来看看具体的效果

如上所示，第一眼是不是觉得做这个东西很难，NO NO NO 其实只要弄懂了其中的原理，做起来真的特别简单，仅仅只是几行代码而已。

先给个提示，这个效果的重点是在border-radius属性上，以及配合CSS动画的旋转效果实现出来的。我们平时用到border-radius是需要让div的方形的四个角 变成圆角，但是平时我们在设置这个属性的时候，会发现，有时候设置出来的角并不是圆形的，而是有点椭圆形状的。我们让这椭圆形的角旋转，然后只取部分让 他显示，就可以模拟出水波荡漾的效果。核心代码如下:

CSS代码

.card { position: relative; width: 300px; height: 350px; background: #fff; border-radius: 2px; box-shadow: 0 2px 15px 3px rgba(0, 0, 0, 0.08); overflow: hidden; } .card::after { content: ''; display: block; width: 100%; height: 100%; background: linear-gradient(to bottom, #0065a8, rgba(221, 238, 255, 0.4) 46%, rgba(255, 255, 255, 0.5)); } .wave { position: absolute; top: 3%; left: 50%; width: 400px; height: 400px; margin-top: -200px; margin-left: -200px; background: #0af; border-radius: 40%; opacity: .4; animation: shift 3s infinite linear; } .wave.w2 { background: yellow; opacity: .1; animation: shift 7s infinite linear; } .wave.w3 { animation: shift 5s infinite linear; background: crimson; opacity: 0.1; } @keyframes shift { from { transform: rotate(360deg); } }

HTML内容

<div class="card"> <div class="wave w1"></div> <div class="wave w2"></div> <div class="wave w3"></div> </div>

在这里，我们叠加了三个波浪，这样看起来更有层次感。效果更加的好。三个叠加的波浪各有不同的颜色，加上透明度让波浪只在顶端进行显示，并给每个波浪 设置不同的动画持续时间。最后就出来了这个层次分明的水波荡漾效果。
