offset-path控制图形沿不规则路线移动
02 November 2017
SVG

今天介绍一个简单但是非常实用的效果,通过SVG方式实现的图形的沿不规则路线移动的效果。话不多说，先来看看demo效果。

重点看得就是这个图片的移动效果，后面红色的移动路径是我另外加了一个svg来显示的。我们来看看代码

.offset{ position: absolute; offset-path: path("M10,80 q100,120 120,20 q140,-50 160,0"); animation: move 3s linear infinite; } @keyframes move { 100% { offset-distance: 100%;} } <img src="" class="offset"> <svg width="280" height="150"> <path d="M10,80 q100,120 120,20 q140,-50 160,0"/> </svg>

代码其实很简单，就是给img的class添加一个offset-path路径和关键帧动画。它就会按照path设置的路径移动了。path的路径设置是跟svg一样的，不懂的同学可以先去了解一下svg路径的原理。

offset-path还有一个比较实用的属性offset-rotation，你可以设置角度值让这个图片的方向进行旋转，或者设置为reverse让这个图片上下颠倒。

这个实现方式很简单，不需要加什么东西。但是在实际的项目中还是不能用，因为这个兼容性特别的差，基本只有chrome以及opera浏览器可以使用，在开发中还是推荐使用SVG SMIL animation来 实现这个沿不规则路径移动的效果。
