也废话不多说，先看看效果。如下所示

当你的鼠标移动到这个div的相应位置的时候，这个div的hover事件会向相应的方向移动。而且这个只是css的效果，没有用到js。哈，是不是感觉很神奇。

这个效果的重点其实就只有两个，一个是CSS3的3d效果，还有一个是CSS3的选择器~。

CSS3的3d效果我就不在这边介绍了，也是比较基本的概念。这块只要在div偏转方向的时候，注意一下3d转换的视角就好。 接下来重点介绍一下CSS的选择器~。根据菜鸟教程里的介绍， 比如p~ul是选择p元素之后的每一个ul元素。通过这个选择器，我们就可以通过不同的分支来控制同一个div。下面上代码:

<div class="container"> <a href="" class="a1"></a> <a href="" class="a2"></a> <a href="" class="a3"></a> <a href="" class="a4"></a> <a href="" class="a5"></a> <a href="" class="a6"></a> <a href="" class="a7"></a> <a href="" class="a8"></a> <a href="" class="a9"></a> <div class="card"></div> </div>

container是一个3d舞台。下面的9个a标签以九宫格的形式放在这个div里面。最下面那个card就是我们所看到的那个在转动的div。

.container .a1:hover ~div{ perspective-origin:100% 100%; transform: rotateY(-10deg) rotateX(10deg); } .container .a2:hover ~div{ perspective-origin:50% 100%; transform: rotateX(10deg); } ...

每次鼠标移到相应的a标签上时，都会使card的位置发生相应的转动。就实现的上面那个例子所示的效果。明白了原理之后，其实这个东西也是很简单的，不过重点还是实现这个效果的思想，讲究对于CSS选择器的深入了解！
