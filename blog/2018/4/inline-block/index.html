我所知道的所有inline-block不显示间隔方法
12 April 2018
CSS HTML

inline-block相对于浮动是比较方便的一个行内排序方法，但是inline-block因为空格的存在，是会有一点间隔的，之前我知道几个方法能去掉，今天又发现了好几个不错的办法 所以在这边做一个合集分享一下。

HTML方法

这类方法通过HTML标签的办法实现

1.移除空格

既然是因为空格产生的间隔，那我们就把空格去掉

<div class="test"> <a href="#">测试1</a><a href="#">测试2</a><a href="#">测试2</a> </div>

不过这个方法对于子元素比较多的情况下不是很友好。

2.间隔错位放置或省略方法

这个方法通过HTML的错位放置来实现

兼容IE <div class="test"> <a href="#">测试1</a ><a href="#">测试2</a ><a href="#">测试2</a> </div> <div class="test"> <a href="#">测试1 <a href="#">测试2 <a href="#">测试3</a> </div> HTML5 <div class="test"> <a href="#">测试1 <a href="#">测试2 <a href="#">测试3 </div>

CSS方法

这类方法通过CSS的属性设置来实现，不需要修改HTML。

1.margin负值 .test{ display:inline-block; } .test a{ ... } .test a + a{ margin-left:-5px; }

因为间隔跟上下文的字体大小有关系，而且还需要多加一段css来让除了第一项的内容改变，不推荐使用这个

2.font-size:0 .test{ display:inline-block; font-size:0 } .test a{ font-size:xxPX; }

这个方法基本上可以解决大部分浏览器下inline-block元素之间的间距,推荐大家使用这个

3.letter-spacing .test{ display:inline-block; letter-spacing:-xPX; } .test a{ letter-spacing: xPX; }

这个办法跟margin负值类似，多种浏览器的兼容可能会有问题。

4.word-spacing .test{ display:inline-block; word-spacing:-5PX; } .test a{ word-spacing: 0; }

这个方法也基本可以解决大部分浏览器的问题
